# F1 Insights

> A Formula 1 data dashboard built with React, TypeScript, and TailwindCSS.

F1 Insights is a front-end project designed to explore and visualize real-time and historical Formula 1 data using the public API from **[https://f1api.dev/](https://f1api.dev/)**. The dashboard demonstrates competence in modern web development, focusing on component-driven UI, clean TypeScript architecture, and efficient client-side data consumption.

## Overview

The project provides structured views for drivers, constructors, standings, schedules, and race results. Data is fetched directly from the external API and presented through responsive, modular interface components. The goal is educational: to showcase front-end proficiency and deliver a clear, maintainable codebase suitable for portfolio presentation.

## Technology Stack

* React.js
* TypeScript
* Shadcn/UI
* TailwindCSS
* f1api.dev (external API)

## Features

* Data fetching from the F1 API with typed responses
* Dashboard layout with reusable UI components
* Pages for drivers, constructors, standings, race weekends
* Responsive design with TailwindCSS utility classes
* Organized data access layer for clean component consumption
* Focus on clarity, maintainability, and front-end best practices
